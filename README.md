# Anyelina Digital VE - Landing Page

Esta es la página oficial de **Anyelina Digital VE**, una plataforma de soluciones digitales integrales.

## Características
- **Diseño Responsivo**: Adaptado para móviles, tablets y escritorio.
- **Integración de Redes Sociales**:
  - Feed de Instagram (vía Behold.so).
  - Timeline de Twitter/X.
  - Barra lateral con enlaces directos.
  - Botones flotantes de WhatsApp y TikTok.
- **Animaciones**: Transiciones suaves con `motion`.
- **Formulario de Contacto**: Listo para ser conectado a un servicio de backend.

## Tecnologías Utilizadas
- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Iconos)
- [Motion](https://motion.dev/) (Animaciones)

## Instalación y Desarrollo

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/anyelina-digital-ve.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Despliegue en GitHub Pages

1. Ejecutar el comando de construcción:
   ```bash
   npm run build
   ```
2. El contenido de la carpeta `dist` es el que debe subirse a la rama `gh-pages` o configurarse en GitHub Pages.
